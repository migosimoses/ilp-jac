What is Jac?

Welcome to Jac! Jac is a modern programming language designed around graph-based computation and the Object-Spatial-Paradigm (OSP).

Key Concepts

  • Graph-Based: Programs are built using nodes and edges
  • Walkers: Mobile agents that traverse graphs
  • Spatial: Data is organized in space, not sequentially
  • byLLM: Built-in AI integration for intelligent programs

Your First Program

with entry { print("Hello from Jac!"); }

Why Jac?

Jac excels at:

  • Knowledge graphs and semantic networks
  • Social network analysis
  • Recommendation systems
  • AI-integrated applications

Nodes and Edges

Nodes and edges are the fundamental building blocks of Jac programs.

What are Nodes?

A node is a data structure that represents a vertex in a graph. Each node can hold data (attributes).

node Person { has name: str; has age: int; has email: str; }

What are Edges?

An edge is a connection between nodes that can have its own attributes.

edge knows { has since_year: int; has strength: float; }

Building a Graph

Combine nodes and edges to create graph structures:

node Person { has name: str; } edge knows { has years: int; } with entry { let person1 = spawn Person(name="Alice"); let person2 = spawn Person(name="Bob"); person1 --> knows(years=5) --> person2; }

Walkers and Traversal

A walker is a mobile entity that traverses a graph, performing operations at each node.

What are Walkers?

Walkers enable:

  • Graph traversal and searching
  • Data processing and transformation
  • Pattern matching in complex structures
  • Building algorithms on graphs

Basic Walker

walker Visitor { can visit { print("Visiting node"); } }

Traversing Edges

Walkers can navigate along edges:

walker Explorer { can explore { print("At node"); take --> node { explore(); } } }
